## Overview

**Brick Breaker GenZ** is a 48-hour game jam project built in **Unity 3D** for a Playtika-sponsored event under the theme *“Reinventing the Classics.”*

The project reimagines the traditional brick breaker formula by placing the gameplay inside a fully navigable 3D space, creating a more dynamic and immersive arcade experience.

Despite the tight development timeframe, the game delivers a complete gameplay loop, interactive mechanics, and a few hidden easter eggs for players to discover.

## My Contribution

- Adapted classic brick breaker logic into a 3D environment
- Designed gameplay flow and difficulty balancing
- Integrated UI and feedback systems
